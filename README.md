# Capitulo 1
Aqui nos explican como internet funciona como una infraestructura global de redes, nos dicen que internet funciona gracias a diversos medios como cables de fibra óptica, transmisiones inalámbricas y enlaces satelitales, esto nos permite que todo aquello que realizamos en linea, como estar en redes sociales, jugar videojuegos en lines se conecte a traves de redes locales.

Encontramos algunos terminos claves sobre el funcionamiento, tales como:
# Redes SOHO (Small Office/Home Office): 
se utilizan para oficinas pequeñas o el hogar.
# Redes Empresariales: 
redes un poco mas grandes utilizadas para publicidad o ventas, suelen ser mas economicas.
# Redes Mundiales:
Se conectan miles de redes locales en todo el planeta.
A eso podemos llamarlo como la clasificación de redes por tamaño.

En este capitulo también nos habla sobre la diversidad de dispositivos indicando que hoy en día hay una gran variedad como por ejemplo los dispositivos moviles que en este caso se incluye, telefonos inteligentes, gps, tablet, etc, también se encuentra la conexión sistemas de seguridad, iluminación y controles de climatización,  los automoviles inteligentes que acceden a mapas o contenidos multimedia.
